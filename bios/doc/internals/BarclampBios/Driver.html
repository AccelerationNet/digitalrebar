- (Boolean) commit

Commit the new settings. This may cause the node to require a reboot..

Parameters:

  • s (Array<BarclampBios::Setting>) —

    The settings that should be committed. All settings in the array must have been returned by a previous call to settings for this driver. An exception will be raised if this is not the case. Additionaly, an exception will be raised if we are unable to commit changes for some reason.

Returns:

  • (Boolean) —

    Whether or not the node needs to be rebooted.
